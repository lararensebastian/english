# English Deep Practice (GitHub Pages)

En enkel offline-vänlig GitHub Pages-sida för IM-elever (A2–B1) med moduler som tar 60–90 min/st. Elever kan jobba 3–4 timmar genom att göra 2–3 moduler.

## Publicera på GitHub Pages
1. Skapa ett nytt repo på GitHub (t.ex. `english-deep-practice`).
2. Ladda upp filerna i denna mapp (index.html, style.css, app.js, data.json).
3. Gå till **Settings → Pages**.
4. Välj **Deploy from a branch**.
5. Välj branch `main` och folder `/ (root)`.
6. Spara. Din sida finns sedan på GitHub Pages-URL:en.

## Redigera/bygga ut
- Allt innehåll ligger i `data.json`.
- Lägg till fler moduler genom att kopiera en modul-objektstruktur och ge unik `id`.

## Tips i klassrummet
- Säg: “Jobba 2 moduler + skrivdelar = godkänt pass”.
- Elever kan trycka Reset allt om de vill börja om.
